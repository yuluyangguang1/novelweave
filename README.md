# NovelWeave · 织文

AI 驱动的网文创作辅助工具。

## 定位
不是"一键生成"的玩具，而是**严肃的写作工作台**。帮助网文作者：
- 管理庞大的世界观（角色/势力/地点）
- 维护长篇一致性（防止人设崩塌、前文矛盾）
- 辅助日常写作（续写卡壳、润色、风格统一）

## 核心原则
- **作者掌控** — AI 只提供建议，作者决定最终文字
- **一致性优先** — 维护角色设定、世界规则、剧情线
- **写作流保护** — 任何 AI 辅助都不打断作者的沉浸感

## 技术路线
- 纯前端（零服务器，打开即）
- BYOK（用户自带 API Key，浏览器直连 LLM）
- IndexedDB 本地存储（支持大文本）
- PWA 可安装
